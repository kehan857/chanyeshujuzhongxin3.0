# 朋友圈营销 (Moments Marketing) 设计规格

## 页面概述
朋友圈营销模块包含定时发朋友圈和AI自动点赞评论两个核心功能，帮助销售团队维护客户关系和提升品牌曝光。

## 移动端适配要点

### 朋友圈编辑器
- 图片上传改为单张逐个上传
- 文本编辑器支持触屏优化
- 时间选择器适配移动端滚轮

### AI配置简化
- 复杂配置项折叠显示
- 关键开关和基础设置优先显示
- 高级配置通过展开查看

## 特殊交互设计

### 实时预览
- 朋友圈内容实时预览效果
- 支持不同设备尺寸预览
- 字数和媒体限制实时提示

### 批量操作
- 支持批量选择历史任务
- 批量修改发布时间
- 批量复制到新任务

### 智能推荐
- 根据历史数据推荐最佳发布时间
- 内容相似度检测避免重复
- AI生成的评论模板推荐
